我们在项目中想要直接通过 ajax 的方式来获取数据，但是操作的时候你会发现失败了，原因就是 javascript 的同源策略，不允许跨域请求，所以我们得到了以下的错误信息：

Access to XMLHttpRequest at ‘localhost:3000/news’ from origin ‘http://localhost:8080’ has been blocked by CORS policy: Cross origin requests are only supported for protocol schemes: http, data, chrome, chrome-extension, https.
